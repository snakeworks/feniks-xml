# Feniks XML
Feniks gives you the ability to write XML to describe scenes in Godot.

# Example
```xml
<HBoxContainer layout_mode="1" anchors_preset="1">
	<Label text="Hello World!"/>
	<Label text="Hello World!"/>
	<Label text="Hello World!"/>
	<VBoxContainer>
		<Control>
			<RichTextLabel text="Indented rich text!"/>
		</Control>
	</VBoxContainer>
</HBoxContainer>
```
